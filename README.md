## Getting Started
To get started, simply clone this repository to your local machine:
```bash
  git clone https://github.com/DuyThaiddt/JavaScript_Learning
```
Once you have cloned the repository, you can start exploring the different directories and files to see the code I have written for different topics in JavaScript.
## Prerequisites
To run the JavaScript code in this repository, you will need to have Node.js and a text editor installed on your computer.
